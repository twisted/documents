twisted.words.protocols.irc.IRCClient.msg(self, user, message, length=None) method documentation

Part of twisted.words.protocols.irc.IRCClient View Source

Send a message to a user or channel.

The message will be split into multiple commands to the server if:

  • The message contains any newline characters
  • Any span between newline characters is longer than the given line-length.
ParametersuserUsername or channel name to which to direct the message. (type: str)
messageText to send. (type: str)
lengthMaximum number of octets to send in a single command, including the IRC protocol framing. If None is given then IRCClient._safeMaximumLineLength is used to determine a value. (type: int)
