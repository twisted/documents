twisted.runner.inetdtap.Options(usage.Options) class documentation

Part of twisted.runner.inetdtap View Source (View In Hierarchy)

To use it, create a file named `sample-inetd.conf` with:

8123 stream tcp wait some_user /bin/cat -

You can then run it as in the following example and port 8123 became an echo server.

twistd -n inetd -f sample-inetd.conf
