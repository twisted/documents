An alternate interface to implement for handling message delivery.

It is useful to implement this interface instead of IMessageDelivery directly because it allows the implementor to distinguish between different messages delivery over the same connection. This can be used to optimize delivery of a single message to multiple recipients, something which cannot be done by IMessageDelivery implementors due to their lack of information.
