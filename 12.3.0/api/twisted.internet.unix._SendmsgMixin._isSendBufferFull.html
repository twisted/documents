t.i.u._._isSendBufferFull(self) : method documentation

Part of twisted.internet.unix._SendmsgMixin 0 View Source

Determine whether the user-space send buffer for this transport is full or not.

This extends the base determination by adding consideration of how many file descriptors need to be sent using sendmsg.send1msg. When there are more than self._fileDescriptorBufferSize, the buffer is considered full.

ReturnsTrue if it is full, False otherwise.
