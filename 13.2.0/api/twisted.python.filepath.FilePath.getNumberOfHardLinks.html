t.p.f.F.getNumberOfHardLinks(self) : method documentation

Part of twisted.python.filepath.FilePath View Source

Retrieves the number of hard links to the file.

This count keeps track of how many directories have entries for this file. If the count is ever decremented to zero then the file itself is discarded as soon as no process still holds it open. Symbolic links are not counted in the total.

Returnsthe number of hard links to the file (type: int)
RaisesNotImplementedErrorif the platform is Windows, since Windows doesn't maintain a link count for directories, and os.stat does not set st_nlink on Windows anyway.
