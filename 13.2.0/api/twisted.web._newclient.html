An HTTP 1.1 client.

The way to use the functionality provided by this module is to:

  • Connect a HTTP11ClientProtocol to an HTTP server
  • Create a Request with the appropriate data
  • Pass the request to HTTP11ClientProtocol.request
  • The returned Deferred will fire with a Response object
  • Create a IProtocol provider which can handle the response body
  • Connect it to the response with Response.deliverBody
  • When the protocol's connectionLost method is called, the response is complete. See Response.deliverBody for details.

Various other classes in this module support this usage:

  • HTTPParser is the basic HTTP parser. It can handle the parts of HTTP which are symmetric between requests and responses.
  • HTTPClientParser extends HTTPParser to handle response-specific parts of HTTP. One instance is created for each request to parse the corresponding response.
