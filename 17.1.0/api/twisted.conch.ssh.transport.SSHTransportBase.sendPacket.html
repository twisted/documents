twisted.conch.ssh.transport.SSHTransportBase.sendPacket(self, messageType, payload) method documentation

Part of twisted.conch.ssh.transport.SSHTransportBase View Source

Sends a packet. If it's been set up, compress the data, encrypt it, and authenticate it before sending. If key exchange is in progress and the message is not part of key exchange, queue it to be sent later.

ParametersmessageTypeThe type of the packet; generally one of the MSG_* values. (type: int)
payloadThe payload for the message. (type: str)
