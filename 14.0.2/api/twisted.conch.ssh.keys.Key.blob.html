Part of twisted.conch.ssh.keys.Key View Source

Return the public key blob for this key. The blob is the over-the-wire format for public keys:

RSA keys:

   string  'ssh-rsa'
   integer e
   integer n

DSA keys:

   string  'ssh-dss'
   integer p
   integer q
   integer g
   integer y
Returns (type: str)
