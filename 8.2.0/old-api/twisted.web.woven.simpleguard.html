Module t.w.w.simpleguard

Part of twisted.web.woven View Source

A simple guard framework for implementing web sites that only need 'Anonymous' vs 'Logged on' distinction, but nothing more.

If you need
  • multiple levels of access, or
  • multiple-interface applications, or
  • anything else more complex than 'Logged on' and 'Not logged on'
you need to use twisted.web.woven.guard directly.
