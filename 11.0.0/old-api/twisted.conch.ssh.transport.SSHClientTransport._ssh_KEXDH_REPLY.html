t.c.s.t.S._ssh_KEXDH_REPLY(self, packet) : method documentation

Part of twisted.conch.ssh.transport.SSHClientTransport View Source

Called to handle a reply to a diffie-hellman-group1-sha1 key exchange message (KEXDH_INIT).

Like the handler for KEXDH_INIT, this message type has an overlapping value. This method is called from ssh_KEX_DH_GEX_GROUP if that method detects a diffie-hellman-group1-sha1 key exchange is in progress.

Payload:

   string serverHostKey
   integer f (server Diffie-Hellman public key)
   string signature

We verify the host key by calling verifyHostKey, then continue in _continueKEXDH_REPLY.
