t.p.b.Int32StringReceiver(IntNStringReceiver) : class documentation

Part of twisted.protocols.basic View Source View In Hierarchy

A receiver for int32-prefixed strings.

An int32 string is a string prefixed by 4 bytes, the 32-bit length of the string encoded in network byte order.

This class publishes the same interface as NetstringReceiver.
