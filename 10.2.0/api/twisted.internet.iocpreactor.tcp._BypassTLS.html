Part of twisted.internet.iocpreactor.tcp View Source View In Hierarchy

_BypassTLS is used as the transport object for the TLS protocol object used to implement startTLS. Its methods skip any TLS logic which startTLS enables.
Instance Variables_connectionA Connection which TLS has been started on which will be proxied to by this object. Any method which has its behavior altered after startTLS will be skipped in favor of the base class's implementation. This allows the TLS protocol object to have direct access to the transport, necessary to actually implement TLS.
