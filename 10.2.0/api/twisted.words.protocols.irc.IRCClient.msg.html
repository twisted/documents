t.w.p.i.I.msg(self, user, message, length=MAX_COMMAND_LENGTH) : method documentation

Part of twisted.words.protocols.irc.IRCClient View Source

Send a message to a user or channel.

The message will be split into multiple commands to the server if:
  • The message contains any newline characters
  • Any span between newline characters is longer than the given line-length.
ParametersuserThe username or channel name to which to direct the message. (type: str )
messageThe text to send. (type: str )
lengthThe maximum number of octets to send in a single command, including the IRC protocol framing. If not supplied, defaults to MAX_COMMAND_LENGTH. (type: int )
