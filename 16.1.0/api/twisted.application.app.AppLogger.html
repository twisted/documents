def start(self, application): (source)

Initialize the global logging system for the given application.

If a custom logger was specified on the command line it will be used. If not, and an ILogObserver component has been set on application, then it will be used as the log observer. Otherwise a log observer will be created based on the command-line options for built-in loggers (e.g. --logfile).
