I hold the lowest-level Resource class and related mix-in classes.
Class RenderMixin Mix-in class for iweb.IResource which provides a dispatch mechanism for handling HTTP methods.
Class Resource An iweb.IResource implementation with some convenient mechanisms for locating children.
Class PostableResource A Resource capable of handling the POST request method.
Class LeafResource A Resource with no children.
Class RedirectResource A LeafResource which always performs a redirect.
Class WrapperResource An iweb.IResource implementation which wraps a RenderMixin instance and provides a hook in which a subclass can implement logic that is called before request processing on the contained Resource.
