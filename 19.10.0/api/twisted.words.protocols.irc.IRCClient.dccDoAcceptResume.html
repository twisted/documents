twisted.words.protocols.irc.IRCClient.dccDoAcceptResume(self, user, file, port, resumePos) method documentation

Part of twisted.words.protocols.irc.IRCClient View Source

Called when a client has verified and accepted a DCC resume request made by us. By default it will do nothing.

ParametersuserThe hostmask of the user who has accepted the DCC resume request. (type: bytes)
fileThe name of the file to resume the transfer of. (type: bytes)
portAn integer representing the port of the accepting user. (type: int)
resumePosThe position in the file from where the transfer should resume. (type: int)
