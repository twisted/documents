twisted.cred.credentials.ISSHPrivateKey(ICredentials) interface documentation

Part of twisted.cred.credentials View Source (View In Hierarchy)

Known implementations: twisted.cred.credentials.SSHPrivateKey

ISSHPrivateKey credentials encapsulate an SSH public key to be checked against a user's private key.

Instance Variable username The username associated with these credentials. (type: bytes)
Instance Variable algName The algorithm name for the blob. (type: bytes)
Instance Variable blob The public key blob as sent by the client. (type: bytes)
Instance Variable sigData The data the signature was made from. (type: bytes)
Instance Variable signature The signed data. This is checked to verify that the user owns the private key. (type: bytes or None)
