twisted.words.protocols.irc.IRCClient.dccDoSend(self, user, address, port, fileName, size, data) method documentation

Part of twisted.words.protocols.irc.IRCClient View Source

Called when I receive a DCC SEND offer from a client.

By default, I do nothing here.

ParametersuserThe hostmask of the requesting user. (type: bytes)
addressThe IP address of the requesting user. (type: bytes)
portAn integer representing the port of the requesting user. (type: int)
fileNameThe name of the file to be transferred. (type: bytes)
sizeThe size of the file to be transferred, which may be -1 if the size of the file was not specified in the DCC SEND request. (type: int)
dataA 3-list of [fileName, address, port]. (type: list)
