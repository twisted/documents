Part of twisted.mail.tap.Options View Source

Add a POP3 port listener on the specified endpoint.

You can listen on multiple ports by specifying multiple --pop3 options. For backwards compatibility, a bare TCP port number can be specified, but this is deprecated. [SSL Example: ssl:8995:privateKey=mycert.pem] [default: tcp:8110]
