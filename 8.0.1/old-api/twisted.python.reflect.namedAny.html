Function t.p.r.namedAny(name):

Part of twisted.python.reflect View Source

Retrieve a Python object from the global Python module namespace, by its fully qualified name. The first part of the name, that describes a module, will be discovered and imported.
Parametersnamethe fully qualified name of a Python object, which is a dot-separated list of python objects accessible via a name. This includes packages, modules, and any Python object which has attributes. For example, a fully-qualified name of this object is 'twisted.python.reflect.namedAny'. (type: str )
Returnsthe Python object identified by 'name'.
RaisesValueErrorif the top level dotted name that is passed is not a valid Python identifier, or the top level dotted name that is passed is not a valid python module.
AttributeErrorif an attribute of an object along the way cannot be accessed, or a module along the way is not found.
ImportErrorif any module involved cannot be imported for some reason.
