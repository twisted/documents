Method t.p.l.L.msg(self, *message, **kw):

Part of twisted.python.log.LogPublisher View Source

Log a new message.

For example: 


>>> log.msg('Hello, world.')
In particular, you MUST avoid the forms: 


>>> log.msg(u'Hello, world.')
>>> log.msg('Hello ', 'world.')
These forms work (sometimes) by accident and will be disabled entirely in the future.
