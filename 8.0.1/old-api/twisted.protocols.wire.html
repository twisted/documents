Module t.p.wire

Part of twisted.protocols View Source

Implement standard (and unused) TCP protocols.

These protocols are either provided by inetd, or are not provided at all.
Class Echo As soon as any data is received, write it back (RFC 862)
Class Discard Discard any received data (RFC 863)
Class Chargen Generate repeating noise (RFC 864)
Class QOTD Return a quote of the day (RFC 865)
Class Who Return list of active users (RFC 866)
Class Daytime Send back the daytime in ASCII form (RFC 867)
Class Time Send back the time in machine readable form (RFC 868)
