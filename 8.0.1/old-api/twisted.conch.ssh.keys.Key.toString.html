Method t.c.s.k.K.toString(self, type, extra=None):

Part of twisted.conch.ssh.keys.Key View Source

Create a string representation of this key. If the key is a private key and you want the represenation of its public key, use .public().toString(). type maps to a _toString_* method. The extra paramater allows passing data to some of the method. For public OpenSSH keys, it represents a comment. For private OpenSSH keys, it represents a passphrase.
