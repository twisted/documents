t.i.t.L.running : instance variable documentation

Part of twisted.internet.task.LoopingCall

A flag which is True while f is scheduled to be called (or is currently being called). It is set to True when start is called and set to False when stop is called or if f raises an exception. In either case, it will be False by the time the Deferred returned by start fires its callback or errback. (type: bool)
