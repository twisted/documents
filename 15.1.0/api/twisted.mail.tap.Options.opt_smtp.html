Add an SMTP port listener on the specified endpoint.

You can listen on multiple ports by specifying multiple --smtp options. For backwards compatibility, a bare TCP port number can be specified, but this is deprecated. [SSL Example: ssl:8465:privateKey=mycert.pem] [default: tcp:8025]
