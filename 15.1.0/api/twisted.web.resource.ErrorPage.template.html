t.w.r.E.template : instance variable documentation

Part of twisted.web.resource.ErrorPage

A native string which will have a dictionary interpolated into it to generate the response body. The dictionary has the following keys:
  • "code": The status code passed to ErrorPage.__init__.
  • "brief": The brief description passed to ErrorPage.__init__.
  • "detail": The detailed description passed to ErrorPage.__init__.
