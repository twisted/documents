t.p.b.P._DOP_EXPRESSIONS : class variable documentation

Part of twisted.positioning.base.PositionError

A mapping of DOP types (C[hvp]dop) to a list of callables that take self and return that DOP type, or raise TypeError. This allows a DOP value to either be returned directly if it's know, or computed from other DOP types if it isn't. (type: dict of str to callables)
