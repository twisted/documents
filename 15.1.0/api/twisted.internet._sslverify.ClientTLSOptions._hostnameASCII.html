t.i._.C._hostnameASCII : instance variable documentation

Part of twisted.internet._sslverify.ClientTLSOptions

The hostname, as transcoded into IDNA ASCII-range unicode code points. This is pre-transcoded because the service_identity package is rather strict about requiring the idna package from PyPI for internationalized domain names, rather than working with Python's built-in (but sometimes broken) IDNA encoding. ASCII values, however, will always work. (type: unicode)
