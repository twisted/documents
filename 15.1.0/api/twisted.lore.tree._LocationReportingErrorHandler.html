t.l.t._LocationReportingErrorHandler(ErrorHandler) : class documentation

Part of twisted.lore.tree View Source View In Hierarchy

Define a SAX error handler which can report the location of fatal errors.

Unlike the errors reported during parsing by other APIs in the xml package, this one tries to mismatched tag errors by including the location of both the relevant opening and closing tags.
