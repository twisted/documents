t.w.p.j.c.X.jid : instance variable documentation

Part of twisted.words.protocols.jabber.client.XMPPAuthenticator

Jabber ID to authenticate with. This may contain a resource part, as a suggestion to the server for resource binding. A server may override this, though. If the resource part is left off, the server will generate a unique resource identifier. The server will always return the full Jabber ID in the resource binding step, and this is stored in this instance variable. (type: jid.JID)
