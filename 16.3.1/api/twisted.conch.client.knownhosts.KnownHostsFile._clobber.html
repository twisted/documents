twisted.conch.client.knownhosts.KnownHostsFile._clobber instance variable documentation

Part of twisted.conch.client.knownhosts.KnownHostsFile
A flag indicating whether the current contents of the save path will be disregarded and potentially overwritten or not. If True, this will be done. If False, entries in the save path will be read and new entries will be saved by appending rather than overwriting. (type: bool)
