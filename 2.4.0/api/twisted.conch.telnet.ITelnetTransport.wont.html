Function t.c.t.I.wont(self, option):

Part of twisted.conch.telnet.ITelnetTransport

Indicate that we will stop performing the given option.

Returns a Deferred that fires with True when the peer acknowledges we have stopped performing this option. If the option is already disabled locally, the Deferred will fail with AlreadyDisabled. If negotiation regarding this option is already in progress, the Deferred will fail with AlreadyNegotiation.

Note: It is currently possible that this Deferred will never fire, if the peer never responds, or if the peer believes the option to already be disabled.
