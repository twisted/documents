Function t.p.p.getPluginFileList(debugInspection=None, showProgress=None):

Part of twisted.python.plugin

Find plugin.tml files in subdirectories of paths in sys.path
ParametersdebugInspectionIf not None, this is invoked with strings containing debug information about the loading process. If it is any other true value, this debug information is written to stdout (This behavior is deprecated). (type: None or a callable taking one argument )
showProgressIf not None, this is invoked with floating point values between 0 and 1 describing the progress of the loading process. If it is any other true value, this progress information is written to stdout. (This behavior is deprecated). (type: None or a callable taking one argument. )
ReturnsA list of the plugin.tml files found. (type: list of str )
