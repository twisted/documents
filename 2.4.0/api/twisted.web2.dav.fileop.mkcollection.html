Function t.w.d.f.mkcollection(filepath):

Part of twisted.web2.dav.fileop

Perform a MKCOL on the given filepath.
Parametersfilepaththe FilePath of the collection resource to create.
Returnsa response or response code. If the operation succeeds, the return value will be responsecode.CREATED if the destination already exists or responsecode.NO_CONTENT if the destination was created by the MKCOL.
