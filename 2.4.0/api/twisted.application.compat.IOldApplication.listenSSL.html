Function t.a.c.I.listenSSL(self, port, factory, ctxFactory, backlog=50, interface=''):

Part of twisted.application.compat.IOldApplication

Add a service that connects a given protocol factory to the port.

The connection is a SSL one, using contexts created by the context factory.
Parametersporta port number on which to listen
factorya twisted.internet.protocol.ServerFactory instance
contextFactorya twisted.internet.ssl.ContextFactory instance
backlogsize of the listen queue
interfacethe hostname to bind to, defaults to '' (all)
