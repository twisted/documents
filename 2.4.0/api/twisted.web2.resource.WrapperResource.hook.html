Function t.w.r.W.hook(self, request):

Part of twisted.web2.resource.WrapperResource

Override this method in order to do something before passing control on to the wrapped resource's renderHTTP and locateChild methods.
ReturnsNone or a Deferred. If a deferred object is return, it's value is ignored, but renderHTTP and locateChild are chain onto the deferred as callbacks.
