Function t.c.s.f.F.makeDirectory(self, path, attrs):

Part of twisted.conch.ssh.filetransfer.FileTransferClient

Make a directory.

path is the name of the directory to create as a string. attrs is a dictionary of attributes to create the directory with. It's meaning is the same as the attrs in the openFile method.

This method returns a Deferred that is called back when it is created.
