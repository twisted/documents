   >>> doc=microdom.parseString('<a>1<b>2<c>3</c>4</b></a>')
   >>> gatherTextNodes(doc.documentElement)
   '1234'
With dounescape=1, also convert entities back into normal characters.
Returnsthe gathered nodes as a single string (type: str )
