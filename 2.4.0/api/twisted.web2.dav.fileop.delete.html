Function t.w.d.f.delete(uri, filepath, depth='infinity'):

Part of twisted.web2.dav.fileop

Perform a DELETE operation on the given URI, which is backed by the given filepath.
Parametersfilepaththe FilePath to delete.
depththe recursion Depth for the DELETE operation, which must be "infinity".
Returnsa response or response code. If depth is not "infinity", responsecode.BAD_REQUEST is returned. If the operation succeeds, the return value will be responsecode.NO_CONTENT.
