Function t.i.i.I.loseWriteConnection(self):

Part of twisted.internet.interfaces.ITCPTransport

Half-close the write side of a TCP connection.

If the protocol this is attached to implements IHalfCloseableProtocol, it will get notified when the operation is done. When closing write connection, as with loseConnection this will only happen when buffer has emptied and there is no registered producer.
