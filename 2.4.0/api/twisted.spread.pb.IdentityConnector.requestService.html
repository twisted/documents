Function t.s.p.I.requestService(self, serviceName, perspectiveName=None, client=None):

Part of twisted.spread.pb.IdentityConnector

Request a perspective on the specified service. This will authenticate against the server as well if requestLogin hasn't already been called.
ParametersserviceNameThe name of the service to obtain a perspective for. (type: string )
perspectiveNameIf specified, the name of the perspective to obtain. Otherwise, default to the name of the identity. (type: string )
clientThe client object to attach to the perspective.
ReturnsA deferred which will receive a callback with the perspective. (type: twisted.internet.defer.Deferred )
