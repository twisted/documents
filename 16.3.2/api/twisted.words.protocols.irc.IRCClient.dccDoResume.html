twisted.words.protocols.irc.IRCClient.dccDoResume(self, user, file, port, resumePos) method documentation

Part of twisted.words.protocols.irc.IRCClient View Source

Called when a client is trying to resume an offered file via DCC send. It should be either replied to with a DCC ACCEPT or ignored (default).

ParametersuserThe hostmask of the user who wants to resume the transfer of a file previously offered via DCC send. (type: bytes)
fileThe name of the file to resume the transfer of. (type: bytes)
portAn integer representing the port of the requesting user. (type: int)
resumePosThe position in the file from where the transfer should resume. (type: int)
